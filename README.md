# G2TMPL

Utilities for GRIB2 templates

Code Manager: Boi Vuong


### Prerequisites

Compilers: GNU | Intel | Clang | AppleClang 


### Installing

```
mkdir build
cd build
cmake -DCMAKE_INSTALL_PREFIX=/path/to/install /path/to/NCEPLIBS-g2tmpl
make -j2
make install
```
